# 📚 منصة الاختبارات الذكية - Arabic Smart Quiz Platform

نظام تعلم عربي تفاعلي ذكي مبني بـ Next.js 14

## 🚀 تشغيل المشروع

```bash
npm install
npm run dev
```
ثم افتح http://localhost:3000

## ✅ الميزات
- ❓ نظام أسئلة تفاعلي مع تصحيح فوري
- 📌 قائمة مراجعة الأسئلة الخاطئة
- 🧪 أوضاع امتحان متعددة (عشوائي، نطاق، مخصص، أخطاء فقط)
- 📊 لوحة إحصائيات كاملة
- 📅 إحصائيات يومية وأسبوعية
- ⚠️ نظام نقاط الضعف الذكي
- 🔥 سلسلة أيام الدراسة (Streak)
- 🏆 نظام الشارات والمستويات
- 💾 حفظ تلقائي في المتصفح (LocalStorage)
- 🌐 واجهة عربية كاملة RTL

## 📁 هيكل المشروع
```
src/
  app/          - Next.js App Router
  components/   - مكونات الواجهة
  data/         - بيانات الأسئلة
  hooks/        - React Hooks
  utils/        - دوال مساعدة
```
